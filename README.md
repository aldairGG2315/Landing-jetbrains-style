# Nexus_Dev: Landing de Alto Rendimiento 

Este proyecto es una muestra técnica de desarrollo frontend modular, enfocada en la implementación de interfaces animadas y principios de diseño moderno.

###  Stack Tecnológico
- **Core:** React 18 + Vite.
- **Estilos:** Tailwind CSS (Arquitectura Utility-First).
- **Animaciones:** Framer Motion (Interpolación de estados y AnimatePresence).
- **Iconografía:** Lucide React.

###  Características Técnicas
- **Glassmorphism:** Implementación de filtros de desenfoque de fondo (backdrop-filter).
- **Morphing Text:** Sistema de transición de texto con gestión de intervalos de tiempo.
- **Liquid UI:** Fondos animados optimizados para GPU.
